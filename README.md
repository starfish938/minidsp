# DSPLib

## 项目简介

本项目使用 C++17 从零实现数字信号处理（DSP）核心算法，包括 FFT、FIR、IIR 及 STFT 等模块。  
项目目标是希望能通过工程化实现加深我对信号处理原理的理解，并结合单元测试验证算法正确性。

---

## 功能模块

### FFT

- Radix-2 Cooley–Tukey 递归实现
- 支持 IFFT 逆变换
- 自动检测 2 的幂长度
- 时间复杂度 O(N log N)
- （基础FFT，有待减少时间复杂度）

### 卷积

- 直接卷积（O(NM)）
- FFT 快速卷积（O(N log N)）
- 自动零填充

### 窗函数

- Hamming 窗
- Hann 窗
- Blackman 窗

### FIR

- 窗函数法设计低通滤波器
- 环形缓冲区实现
- 支持单样本与批量处理

### IIR

- 双线性变换设计一二阶低通和一阶高通
- Direct Form I
- Direct Form II
- 稳定性验证

### STFT

- 帧分割与重叠处理
- 加窗
- 频谱计算
- dB 幅度谱生成

- STFT 时频趋势验证
